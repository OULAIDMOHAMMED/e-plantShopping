# e-plantShopping

e-plantShopping is a React and Redux plant shopping application for Paradise Nursery.

The app includes a landing page, a plant catalog grouped by categories, add-to-cart behavior, cart quantity controls, item removal, and a total cart amount view.

## Project

Paradise Nursery helps users browse indoor plants such as air-purifying, aromatic, insect-repellent, medicinal, and low-maintenance plants, then manage selected items in a shopping cart.
